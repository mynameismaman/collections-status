# Gallary Muslim Web

This is a web for image collection.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can add image .jpg or .png file to `public/images/`. The folder can show all image to page [http://localhost:3000/gallery](http://localhost:3000/gallery).
